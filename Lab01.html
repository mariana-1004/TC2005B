  • 1.- ¿Cuál es la diferencia entre Internet y la World Wide Web?

A diferencia del internet, la World Wide Web es un sistema que funciona a partir del internet que transmite diversos tipos de datos a través del protocolo HTTP

  • 3.- ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
    Indican la acción que se desea ejecutar para un recurso determinado.
    • GET: Se solicita una representación en específico de un recurso. Con este método sólo se puede recuperar datos
    • HEAD: Se pide una respuesta idéntica al de la petición de GET, pero si el cuerpo de la petición
    • POST: Con este se envía una entidad a un recurso en específico, provocando un cambio en el estado del servido
    • PUT: Remplaza todas las representaciones actuales del recurso destino con la carga de la petición
    • PATCH: Se ocupa para realizar modificaciones parciales a un recurso
    • DELETE: Borra el recurso seleccionados

  • 9.- ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

  • 10.- ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?
    HTML5 mejora la organización de las páginas web con elementos más descriptivos y facilita la creación de formularios. Además, permite integrar audio, video y gráficos sin complementos externos, ofrece almacenamiento local y garantiza una experiencia móvil fluida.
  • 11.- ¿Qué componentes de estructura y estilo tiene una tabla?
    Tiene varios componentes de estructura y estilo que se utilizan para organizar y presentar datos de manera tabular.
    • table - define como tal la tabla
    • tr - Cada fila de la tabla se debe de definir con este comenado
    • th - se definen los encabezados de las celdas
    • td - Define las celdas de datos
    • tfoot - es opcional y se ocupa para el pie de la tabla
  • 12.- ¿Cuáles son los principales controles de una forma HTML5?
    • input - Se utiliza para varios tipos de entrada números, texto, correo electrónico, contraseñas
    • textarea - Recibe como entrada texto multilínea
    • select - proporciona una lista desplegable de opciones
    • radio - Permite seleccionar una opción de un conjunto de opciones mutuamente excluyentes.
    • button - Botón para enviar un formulario o activar una acción.
