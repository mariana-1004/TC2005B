24 Preguntas


  1. ¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications)?
    permite actualizar solo partes de una página sin necesidad de recargar todo, lo que hace que la experiencia del usuario sea mucho más rápida y fluida. Permite reducir lo tiempos de carga y también facilita las interacciones en tiempo real, haciendo que las aplicaciones sean más dinámicas y receptivas.

  2. ¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?
    Exposición a ataques de inyección, cross-site scripting (XSS), y cross-site request forgery (CSRF). Las validaciones de seguridad deben hacerse siempre del lado del servidor, ya que las validaciones del lado del cliente pueden ser manipuladas fácilmente por usuarios malintencionados. Aunque las validaciones del lado del cliente pueden mejorar la experiencia del usuario al proporcionar retroalimentación inmediata

  3. ¿Qué es JSON?
    Es un formato de texto ligero y fácil de leer y escribir para el intercambio de datos. JSON se utiliza principalmente para transmitir datos entre un servidor y una aplicación web, siendo una alternativa más simple y eficiente a XML.
