Laboratorio 4
  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    • Direfencias:
    • JavaScript es un lenguaje interpretado y Java es compilado.
    • Java es un lenguaje de programación orientada a objetos puros y JS puede emular la programación orientada a objetos
    • JS es gestionado por ECMAScript y Java por Oracle.
    • Semejanzas:
    • Tienen una sintaxis influenciada por el lenguaje C.
    • Ambos necesitan de un entorno de ejeccuión, para ejecutar el código en diferentes plataformas.
    • Ambos soportan la POO.
  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    1. Date.prototype.getDate() - Devuelve el día del mes de la fecha ingresada.
    2. Date.prototype.getMonth() - Devuelve el mes de la fecha especificada, sólo que empieza a contar desde 0.
    3. Date.prototype.getFullYear() - Retorna el año de la fecha especificada.
    4. Date.prototype.getHours() - Devuelve la hora del día para la fecha especificada.El resultado está en el rango de 0 (12 AM) a 23 (11 PM).
    5. Date.prototype.getTime() - Devuelve el valor numérico dependiendo a la fecha y hora especificadas en milisegundos desde el 1 de enero de 1970 00:00:00 UTC (también conocido como Unix timestamp).
  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
    1. push() - Añade elementos al final del arreglo.
    2. pop() - Elimina el ultimo elemento del arreglo.
    3. shift() - Elimina el primer elemento de un arreglo y desplaza los demás elementos a posiciones más bajas.
    4. unshift() - Añade uno o más elementos al inicio del arreglo.
    5. slice() - Devuelve una copia de una parte del arreglo en un nuevo arreglo, seleccionado desde inicio hasta fin (fin no incluido). El arreglo original no se modifica.
  • ¿Cómo se declara una variable con alcance local dentro de una función?
    Poniendo "let", "var" o "const" antes de poner el nombre de la variable.
  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    Usar variables globales en nuestras funciones puede complicarnos la vida, ya que pueden mezclarse, hacer que nuestro código sea difícil de seguir y mantener, aumentar los riesgos de seguridad.

Material design

Es un lenguaje visual que combina los principios fundamentales del diseño de calidad con las innovaciones de la ciencia y la tecnología. Esta filosofía de diseño surgió en 2014, inicialmente estaba contemplado para el sistema Android, pero con el paso de los años se extendió por toda la web. Material Design está inspirado en papel, es por eso que todo el entorno se construye considerando los atributos del espacio tridimensional: superficies, profundidad, así como luz y sombra.

  1. Superficies: En Material Design todos los elementos tienen profundidad y se pueden mover en el eje "x", en el eje "y" y tiene con profundidades variables a lo largo del eje "Z"
  2. Elevaciones: Se define como la distancia que exite entre los diferentes elementos y superficies. Se mide en el eje "z" desde las partes frontales de cada superficie
  3. Luz y Sombra En la interfaz de usuarios existen dos tipos de luces: Clave (obtiene sombras precisas) y Ambientales (aparecen desde los ángulos, creando sombras más suaves y difusas). El saber utilizar la luz y la sombra, permite generar una coherencia entre el diseño y la experiencia que tiene el usuario
